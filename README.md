# de-express

> `de-express` is a simple way to use [express](http://expressjs.com/) with ES6 decorators.

[![NPM](https://nodei.co/npm/de-express.png)](https://www.npmjs.com/package/de-express)

## Install

```shell
yarn add de-express
# or npm install de-express
```
